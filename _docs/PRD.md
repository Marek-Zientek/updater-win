# PRD - UpdaterWindows - Aktualizacje aplikacji w systemie windows

## 1. Przegląd Projektu

### 1.1 Cel Produktu

Stworzenie prostej aplikacji na systemy windows, które da możliwość prostej aktualizacji aplikacji. Aplikacja powinna mieć także prostą lecz nowoczesną stronę typu landing page prezentującą informacje systemowe, informacje sprzętowe, informacje o oprogramowaniu.

### 1.2 Problem biznesowy

**_Główny problem_** Głównym problemem jest czas, który trzeba poświęcić na aktualziacje oprogramowania. Często użytkownik zapomina do czego tak naprawde jest dana aplikacja i co ona tak naprawde robi. Częstym problemem użytkownika jest jest brak informacji systemowych, czy brak informacji sprzętowych oraz informacji o jakiś zależnościach.

### 1.3 Wartość Biznesowa

**_Główna wartość_** Możliwość rozbudowy o dodatkowe funkcjonalności bez ograniczeń

- Aktualizacja sterowników
- Aktualizacja oprogramowania
- Landing page dla każdej aplikacji lub sterownika
- Landing page dla informacji systemowych czy sprzętowych
- Grafy ze statystykami

**Dodatkowe korzyści:**

- Pełna kontrola nad aplikacją oraz zarządzaniem.

## 2. Użytkownicy docelowi

### 2.1 Użytkownik główny - Klient

**Persona: "Administrator"**

- potrzeby: zarządzanie aplikacją, pełna konfigurowalność
- Oczekiwania: prostota obsługi

### 2.2 Użytkownik trzeci - barber

**Persona: Użytkownik**

- Potrzeby: Widok dostepnych aplikacji oraz sterowników, możliwość aktualizacji
- Oczekiwania: prostota w obsłudze

## 3. Wymagania funcjonalne

### 3.1 System aktualizacji oprogramowania

#### 3.1.1 Rezerwacja z Kontem ( Zarejestrowany użutkownik )

**User Stories:**

- Jako klient chcę szybko zalogować się do systemu, aby sprawdzać aktualności oprogramowania oraz bierzące informacje systemowe.
- Jako klient chcę zobaczyć dostępne aktualizacje do wykonania
- Jako klient chcę wybrać aplikacje które będą aktualizowane
  Jako klient chcę otrzymać natychmiastowe powiadomienia o aktualizacji

**Funkcjonalności**

- Rejestracja konta ( email, hasło, dane podstawowe )
- Login / logout z opcją zapamiętaj mnie
- Dashboard z przyszłymi i przeszłymi aktualizacjami oraz logowaniami
- Wybór co chce lub co mogę zaktualizować
